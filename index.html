Primeiramente, algumas dicas básicas do VsCode que eu utilizo para agilizar a codificação!

  1. Existe um plugin que abre direto no navegador, sem precisar ter q colocar o caminho, chama "OPEN IN BROWSER". Depois que instalar só precisa clicar com o botão direito no html e clicar no mesmo nome.
  2. Se você esquecer de algum comando, tente apertar "CTRL + BARRA DE ESPAÇO", esse comando preenche o restante do comando.
  3. Se precisar repetir o código que acabou de digitar, aperte as teclas "ALT + SHIFT + &↓", ele vai repetir o código que acabou de escrever embaixo, se pre cisar de que seja em cima, só usar ↑!
  4. Para colocar "" ou '' ou (), etc no código, simplesmente selecione todo o código que que deva ser envolvido e pressione o a tecla correspondente ao caracter necessário!
  5. Se precisar escrever algo em baixo mas está no meio de uma frase ou de um código, pressione "CTRL + ENTER", isso fará o cursor passar para a linha abaixo.
  6. Para formatar seu documento, aperte "ALT + SHIFT + F"!
